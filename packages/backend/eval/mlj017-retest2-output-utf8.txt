## Project Phases and Milestones

- (p. 10) **Investigation and Survey Phase (February 2025 - April 2025):** Conduct site mobilization, investigation, surveys, test pits, borings, and conduit tracing, utilizing existing outages and flagging.
- (p. 10) **Design Phase (January 2025 - July 2025):** Complete early work design in three submissions (90%, 100%, RFCD) and general packages starting at 60%, with no public closures.
- (p. 12, p. 4) **Phase 1 (October 2025 - February 2026):** Relocate/install temporary platform utilities, remove canopy rivets, demolish canopy roof and structural steel, and establish ADA boarding areas at local tracks.
- (p. 13, p. 4) **Phase 2 (February 2026 - August 2026):** Complete remaining platform utility relocation, install barricades around elevator/steel columns, perform partial platform slab demolition, install temporary platforms (if needed), and modify platform structural steel.
- (p. 4) **Phase 3 (August 2026 - January 2027):** Deliver and install new platforms.
- (p. 4) **Phase 5 (July 2027 - July 2028):** Drawing shows barricades only on the Jamaica-bound platform.

## NCR Log Data Requirements

- **Date:** Date the NCR was entered into the log (p. 5, 8).
- **NCR No.:** Unique Non-Conformance Report number (p. 5, 8).
- **Brief Description of Non-Conformity:** Summary of the non-conforming item (p. 5, 8).
- **Raised By:** Individual or entity who identified the non-conformity (p. 5, 8).
- **Date of Issuance:** Date the NCR was formally issued (p. 5, 8).
- **Location:** Physical location of the non-conforming item (p. 5, 8).
- **Responsible Party / Subcontractor:** Entity accountable for the non-conformity (p. 5, 8).
- **Disposition:** Proposed resolution for the non-conformity (p. 5, 8).
- **Verification:** Confirmation that the disposition was implemented (p. 5, 8).
- **Initial Status:** Current status of the NCR (p. 5, 8).
- **Date Closed:** Date the NCR was officially closed (p. 5, 8).
